# Prpr::Merged

[Prpr](https://github.com/mzp/prpr) plugin to announce pull request merge.

## Install

```ruby
gem 'prpr-merged'
```

## ENV

 * MERGED_ROOM: room name to announce
 * MERGED_BODY: announce message template
